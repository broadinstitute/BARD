PPM

Codename: ppm

Marvin is able to export PPM image files. The PpmExport module recognizes the common image export options. Examples:

ppm Default settings: 200x200 pixels, white background (or black in 3D).
ppm:w100,#ffff00 100x100 PPM with yellow background.

PpmExport uses the PPM encoder, Copyright © Jef Poskanzer.
