This example demonstrates the available modes of importing structures.

Open molecule
1AID
1PXX
1ETS
Smiles

By clicking on the first radio button, 1AID.pdb is loaded. The input structure is referenced by the relative address "../../../../marvinspace/data/1AID.pdb", which means the file is located relatively to the applet codebase. This is equivalent of the api call 

        document.MSpaceApplet.setMolecule( "../../../../marvinspace/data/1AID.pdb" );

The second option is loading of 1PXX. It is given by absolute URL, but it is slower than loading by a relative address. It can be invoked by 

        document.MSpaceApplet.setMolecule( "http://www.chemaxon.com/marvinspace/data/1PXX.pdb" );

A special way of importing macromolecules is through the RCSB Protein Data Bank. In this case, giving the 4 letter PDB ID is sufficient, but the api call is different: 

        document.MSpaceApplet.setPDB( "1ETS" );

The last example shows that MarvinSpace can directly import structures from a Smiles string. 

        document.MSpaceApplet.setMoleculeString( "[H]C(=O)C([H])([H])C([H])([H])C([H])([H])[H]" );
