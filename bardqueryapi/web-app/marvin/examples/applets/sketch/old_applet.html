MarvinSketch Example - Embedding MarvinSketch applet in HTML pages without the new splash screen

In this example the way to embed a MarvinSketch applet in an HTML page without the new splash screen will be shown. This is the old behaviour of the applet, and the JMSketch applet class will be used to do this.

Simple embedding without splash screen

To support the use of older style Marvin Applets, ChemAxon provides utility functions that help to overcome the problems of working with HTML tags. These utilities are stored in the marvin.js JavaScript file.
As you see in the Simple Embedding example, the msketch_begin function, with two additional parameters will be used.
The fourth parameter is a boolean and it specifies whether the old MarvinSketch Applet without the splash screen will be used, or the new one (true or false).
The live old MarvinSketch applet embedded in the HTML page appears as follows:

To achieve this, the following lines should be included in the HTML page:

<script type="text/javascript" SRC="../../../marvin.js"></script>
<script type="text/javascript">
<!--
msketch_begin("../../..", 540, 480, true, null ); // arguments: codebase, width, height, oldbehaviour, splashscreen
msketch_end();
//-->
</script>

Technical background

The functions above generate the following code, or equivalent with <embed> or <object> tag: 

<applet CODEBASE="../../.." ARCHIVE="jmarvin.jar"
        CODE="JMSketch" WIDTH=540 HEIGHT=480>
<strong>Your browser does not support the applet tag.</strong>
</applet>
