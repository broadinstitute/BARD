MarvinView Example - Background colors, 3D, rotation

All the molecules are shown in 3D now. Other changes:
  • table background color is the same as the page color
  • molecule background is black
  • default mouse drag action is "Rotate in 3D"
  • two molecules are continuously rotated
  • the rendering style is "ball and stick"
