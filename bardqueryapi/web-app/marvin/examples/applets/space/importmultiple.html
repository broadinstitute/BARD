MarvinSpace

This example shows how to set multiple structures by argument.

Multiple structures can be given by the molecules applet parameter.

A special separator caracter can be defined by the molSeparator parameter, the default value is " ". In this example we used the ";" character.

        <param name="molSeparator" value=";">
        <param name="molecules" value="http://www.chemaxon.com/marvinspace/data/1ETS.pdb;
                                       http://www.chemaxon.com/marvinspace/data/1ETR.mol;
                                       http://www.chemaxon.com/marvinspace/data/1ETT.mol">
