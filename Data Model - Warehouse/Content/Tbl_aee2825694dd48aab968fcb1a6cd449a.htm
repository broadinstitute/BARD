Dimension Table Type  Slowly Changing Type 2 
Owner  BBDS 
Definition  
This slowly changing dimension is based solely on RESULT_TYPE_STATUS.  Result_type_status is an amalgam of the source element_status for result_type and stats_modifier.  Changes to result_type and stats_modifier names are implemented through the denormalization procedure p_element.update_element_name.
