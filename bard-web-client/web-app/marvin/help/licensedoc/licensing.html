About ChemAxon Licensing

A ChemAxon software license grants you the legal right to use a piece of ChemAxon software. For each software program you use, you need a license, which is granted to you and documented in the license agreement.

ChemAxon has recently introduced its new license generation system, which will document all relevant information describing the license purchased in a comprehensive and easily handlable form.

The license file should be installed using the ChemAxon License Manager.
Further details can be found at the Frequently asked questions section.

Evaluation Licenses

Free provision of licenses for an evaluation period of 1 month which unlimit functionality of all tools.

Annual Licenses

Annual Licenses offer all functionalities, including product upgrades and support for 1 year.

Perpetual Licenses

A Perpetual License is an unlimited commercial license, with 1 year upgrade and support.

Academic Teaching Licenses

Free ongoing provision of all tools for teaching, including licenses to allow students of the department to use during tuition.

Academic Research Licenses

Recurring 2 year provision of all tools for individual academic researchers.

Academic Web Licenses

Unlimited institutional license to set up chemical communications and structure database storage, management and search for campus wide delivery for teaching and non commercial, academic research. Provision includes:
  • MarvinSketch, MarvinView and MarvinSpace applets
  • All Calculator Plugins - for held content only, users may not perform calculation on structures drawn by site users - instead these users should apply for their own personal Academic Package license to enable this functionality
  • JChem Base/JChem Cartridge
  • Instant JChem - allowing access to remote databases or remote access to your databases, full structure canonicalization options and batch processing of calculator plugins
  • Standardizer
  • Other ChemAxon toolkits can be made available upon request

FreeWeb Licenses

Available for non commercial, freely accessible web pages. The program provides all the technology required to set up chemical communications and structure database storage, management and search capabilities for web servers and pages. Provision includes:
  • MarvinSketch, MarvinView and MarvinSpace applets
  • All Calculator Plugins - for held content only. Users may not perform calculations on structures drawn by site users.
  • JChem Base/JChem Cartridge
  • Standardizer
