Reaction arrow

The following reaction arrows from the CDX file format are imported to Marvin as:

ChemBioDraw Marvin
Solid Single reaction arrow type

Note that files containing multistep reactions are imported with graphical arrows, thus these won't be recognized as single reactions in Marvin.
